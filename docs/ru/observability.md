# Состояние и место на сервере

[English version](../observability.md)

Раз в час лёгкая задача записывает только технические счётчики в `data/health-metrics.jsonl`. Она не
читает тексты сообщений, промпты, vault, ключи или содержимое Workflow.

```bash
iva status
iva doctor
```

Статус показывает память процессов, рестарты, размер и рост Workflow, очередь, свободный диск, inode и
swap. Хранится не больше 744 замеров — примерно 31 день. Файл приватный (`0600`) и не входит в backup.

Первые семь дней собирается baseline без предупреждений. Затем Iva может сообщать о нехватке диска,
swap выше 90%, зависших запусках или росте, способном заполнить диск за неделю. Повторные уведомления
ограничены одним в 24 часа.

Iva никогда автоматически не удаляет Workflow, память, задачи или логи ради освобождения места.
