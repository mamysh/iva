# Провайдеры и стоимость

[English version](../providers.md)

Iva работает на вашем сервере с вашими ключами. Цены, модели и доступность по регионам меняет сам
провайдер, поэтому универсальной месячной цены нет. Перед оплатой проверьте условия и поставьте лимит
расходов, если он поддерживается.

## Модели

| Провайдер | Оплата | Текст | Изображения |
|---|---|---|---|
| **Ollama Cloud** | бесплатные и платные тарифы | по умолчанию `deepseek-v4-pro` | по умолчанию `minimax-m3` |
| **OpenCode Zen** | оплата по тарифу провайдера | доступные модели Zen | зависит от выбранного маршрута |
| **OpenRouter** | оплата по факту | 300+ моделей по slug | отдельная vision-модель |
| **OpenAI через ChatGPT** | существующая Plus/Pro/Team-подписка | доступные плану модели | мультимодальный маршрут той же подписки |

Iva не добавляет наценку. Первые три варианта используют API-ключ; ChatGPT подключается через OAuth:

```bash
iva login
iva config
iva restart
```

Текст и vision — отдельные роли. По умолчанию vision наследует `MODEL_PROVIDER`; опциональный
`VISION_PROVIDER` направляет изображения через другой уже настроенный доступ. Автоматического
переключения на платный провайдер нет.

OpenRouter требует модель с tool/function calling. Мастер делает живую проверку и не принимает неверный
slug или несовместимую модель. Все переменные находятся в [configuration.md](configuration.md).

## Дополнительные расходы

- Голос расшифровывает Deepgram Nova-3; актуальные бесплатные кредиты проверяйте у Deepgram.
- Веб-поиск использует Tavily, Exa, Parallel или Brave. Без ключа поиск честно отключён.
- Гибридный поиск по памяти опционально использует Jina или DeepInfra embeddings.
- Итоговая стоимость: VPS + модель/подписка + выбранные голосовые и поисковые сервисы.

Telegram Bot API не скачивает файлы больше 20 МБ, поэтому длинные видео нужно делить заранее.
