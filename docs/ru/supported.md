# Поддерживаемые версии и ограничения

[English version](../supported.md)

## Поддерживаемая база

- Node.js 24.x.
- Ubuntu 24.04 LTS, Linux x64.
- Локальный Workflow-профиль или opt-in PostgreSQL 17.
- Один владелец и только один активный polling-процесс на bot token.

RC считается проверенным только когда тег `vX.Y.Z-rc.N` совпадает с версией, полная release-матрица
относится к тому же commit, а vision-модель подтверждена ограниченным живым запросом.

## Текущий статус

Для `v0.3.0-rc.6` собираются automated, live-provider и soak evidence; он ещё не является stable. Ollama по умолчанию использует
DeepSeek V4 Pro для текста и MiniMax M3 для изображений. Доступность и цены моделей контролируют
провайдеры и могут меняться независимо от Iva.

## Ограничения

- Telegram userbot — opt-in beta и не входит в stable core.
- PostgreSQL остаётся opt-in; для смены backend по умолчанию нужно отдельное 30-дневное наблюдение.
- ARM64, Windows, macOS, Vercel, Ubuntu 22.04 и Debian пока не входят в stable release-матрицу.
- `main` — канал разработки. После релиза обычные пользователи должны ставить ветку `stable`.
- Production получает только ограниченные проверки после деплоя и не используется для автоматических тестов.
