## [0.1.0] — 2026-06-20

Первый публичный релиз. Личный ИИ-агент с долговременной памятью, который ставится одной командой и
работает там, где подписки на топовых ассистентов недоступны.

### Возможности
- **Установка одной командой** — `curl … | bash` + интерактивный мастер (провайдер, ключи, Telegram, TZ).
- **Telegram через polling** — без домена, вебхука и прокси; заводится на голом VPS без белого IP.
- **Выбор провайдера модели** — Ollama Cloud или OpenCode Zen (оба OpenAI-совместимы, работают с RU-IP),
  рекомендуемая модель DeepSeek-V4-Pro; ключи свои.
- **Голос и видео** — автоматическая расшифровка через Deepgram nova-3 (мультиязык) до попадания к агенту.
- **Древовидная память** — vault из датированных markdown: день → неделя → месяц → год, типизированные
  карточки autograph, оглавление **MOC** и always-on ядро **CORE**; навигатор-карта **MAP** в контексте.
- **Поиск по памяти через grep** — без эмбеддингов и векторных баз; контекст модели остаётся маленьким.
- **Ночные роллапы** — systemd-таймеры сами перерабатывают день в карточки и саммари + проверка здоровья.
- **Инструменты** — веб-поиск, чтение страниц, интерактивный браузер `agent-browser`, задачи, host-shell.
- **Скиллы и MCP** — `morning-digest`, `web-research`, `agent-browser`; скаффолд MCP-подключений.
- **Telegram-команды** — `/help`, `/restart`, `/new`, `/task`, `/tasks`, `/digest`.
- **Безопасность и приватность** — allowlist по Telegram ID (fail-closed); память в отдельном приватном
  git-репо; ключи в `.env`.
- **Настраиваемая личность** — `agent/instructions.md` (по умолчанию русский, кратко).
